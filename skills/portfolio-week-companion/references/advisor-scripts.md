# 顾问话术参考模板

本文件为 `portfolio-week-companion` 的配套参考，供 Xueqi 在撰写模块 13（财经新闻与沟通策略）和模块 14（理财经理沟通贴士）时参考使用。

话术须先经 Sage 合规审核，再嵌入报告。

---

## 一、本周沟通重点（3句话模板）

### 模板结构

```
1. 定调：「本周产品有所[波动/上涨/回调]，[核心资产]端仍[贡献稳定收益/表现坚挺]，[其他资产]端[小幅拖累/有所提振]。」
2. 归因：「净值[变化]主要是受[市场/资产类别]影响，但组合[韧性/分散性]较好，[某资产]起到[缓冲/增厚]作用。」
3. 建议：「建议[继续持有/关注/适当配置]，若客户有闲置资金，当前是较好的[分批配置/定投]时机。」
```

### 上涨行情示例

1. 定调：「本周产品有所上涨，权益端表现较好，债券端继续贡献稳健收益。」
2. 归因：「净值上涨主要得益于A股反弹，组合配置的[具体行业]基金表现亮眼。」
3. 建议：「建议继续持有，若客户有闲置资金，可考虑适当加仓。」

### 下跌行情示例

1. 定调：「本周产品有所回调，债券端仍贡献稳定收益，权益端小幅拖累。」
2. 归因：「净值回调主要是受A股震荡影响，但组合韧性较好，债券端起到缓冲作用。」
3. 建议：「建议继续持有，若客户有闲置资金，当前是较好的分批配置时机。」

### 震荡行情示例

1. 定调：「本周产品整体运行平稳，债券端继续贡献票息收益，权益端小幅波动。」
2. 归因：「市场在[具体事件]影响下窄幅震荡，组合多资产配置起到了平滑波动的作用。」
3. 建议：「建议保持耐心，震荡市正是多资产配置发挥价值的时机。」

## 二、不同客群话术模板

### 2.1 新客户（持仓<1个月）

**核心关切**：对产品不熟悉，对短期波动敏感

```
「您刚买入不久，可能对产品还不太熟悉。这款产品是以[核心资产]为主、适度配置[其他资产]的[产品类型]，
目标是帮您获取比理财高一点的稳健收益。短期小幅波动是正常的，建议您给产品足够长的观察时间。
我们的周度陪伴报告会每周向您更新产品运作情况，有任何疑问随时联系。」
```

### 2.2 盈利老客户（持有6月+，目前盈利）

**核心关切**：想止盈，怕回调

```
「您持有已经超过建议时长，目前也是盈利的，说明产品的策略是有效的。
建议您可以继续把这款产品作为资产配置的底仓，长期持有体验会更好。
如果您担心短期回调，可以考虑[止盈部分/分批止盈]，但不建议全部赎回，
以免错过后续的复利增长机会。」
```

### 2.3 浮亏客户（持仓较短，<3个月）

**核心关切**：焦虑，质疑产品

```
「您目前持仓时间还不长，出现小幅浮亏主要是受短期[市场/板块]波动影响。
产品[核心资产]端仍在稳定贡献收益，等市场回暖后净值会很快修复，建议耐心持有。
历史数据显示，本产品历次短期波动平均在[X]天内修复，您不必过于担心。」
```

### 2.4 浮亏客户（持仓较长，>6个月）

**核心关切**：长期浮亏，信心动摇

```
「您持仓已经比较久了，但目前仍有浮亏，我能理解您的焦虑。
从数据看，当前[权益市场估值/债市收益率]处于相对[低位/合理区间]，
如果这部分资金不急用，可以考虑适当补仓降低成本；
如果已经影响到了您的持有体验，我们可以一起再看看整体资产配置是否需要调整。」
```

### 2.5 高净值客户

**核心关切**：资产保全、家族财富、税务、传承

```
「从您的整体资产配置来看，这款产品定位为[稳健底仓/卫星配置]，
在整个组合中起到[降低波动/增厚收益]的作用。当前[市场判断]，
建议维持现有配置比例，同时可以考虑在[其他资产]上适度增配，
以进一步优化组合的风险收益特征。」
```

### 2.6 机构客户

**核心关切**：业绩基准、合规、信息披露

```
「本周产品[跑赢/跑输]业绩基准[X]个百分点，主要归因于[具体原因]。
底层基金运作正常，无重大异常事件。当前组合[再平衡状态]。
详细的归因分析和底层基金评价报告将另行发送。」
```

## 三、常见客户异议回应

### Q1：「FOF是不是收费太高了？」

```
「FOF确实有两层收费，但您买到的是专业团队为您筛选、调仓、再平衡的一篮子基金组合。
自己买多只基金不仅费时费力，还可能买到风格重复的。FOF帮您解决了选基难、配基难、调仓难的问题，
这些专业服务的价值，长期看是物超所值的。而且我们选择的底层基金多为C类份额或低费率品种，
整体费率在行业中处于合理水平。」
```

### Q2：「为什么不直接买债券基金？」

```
「纯债基金确实波动更小，但长期收益天花板也较低。这款产品在债券底仓基础上，
加入了适度权益和黄金配置，目的是在控制回撤的前提下，争取比纯债更高的长期回报。
从成立以来的数据看，本产品在[大部分/部分]时段跑赢了纯债基金指数，同时回撤控制在[X]%以内，
这就是多资产配置的价值。」
```

### Q3：「持有期太长了，急用钱怎么办？」

```
「持有期的设计，是为了让基金经理有稳定的管理空间，避免短期申赎冲击。
建议您用短期不用的闲钱来配置。如果您对流动性要求很高，
我们可以搭配一些货币基金或短债基金一起配置，形成"核心+卫星"的流动性方案。」
```

### Q4：「为什么大盘涨了组合没涨？」

```
「大盘涨不代表所有资产都涨。本产品是一只[多资产/FOF]产品，配置中包含债券、黄金等
与股市低相关的资产。当股市上涨时，这些资产可能表现平稳，导致组合涨幅不及纯权益产品。
但反过来说，当股市下跌时，这些资产也会起到缓冲作用。这就是『不把所有鸡蛋放在一个篮子里』
的配置逻辑，短期看可能『慢』，长期看更『稳』。」
```

### Q5：「要不要减仓？」

```
「是否减仓取决于您的资金用途和风险承受能力，而不是短期市场波动。
如果这笔钱是[短期要用的/影响生活的]，可以考虑适当减仓；
如果是[长期不用的闲钱/养老/教育金]，建议继续持有，当前[估值/利率]处于[相对合理/低位]区间，
长期配置价值仍然存在。我们可以从您的整体资产配置角度来评估，而不是单看这一只产品。」
```

### Q6：「这只基金为什么跌得多？」

```
「这只基金本周跌幅较大，我们分析主要有[X]个原因：①[市场/行业因素]；②[基金经理调仓]；③[风格切换]。
从Castor的基金评价来看，该基金[长期业绩/风控能力/基金经理稳定性]仍然[良好/需要关注]，
[建议继续持有观察/建议关注后续表现]。如果持续跑输同类，我们会考虑在再平衡时替换。」
```

### Q7：「别人家产品收益更高」

```
「不同产品的定位不同，不能简单比较收益。这款产品的核心目标是[稳健增值/控制回撤/多资产配置]，
而不是追求短期高收益。如果您对比的是[纯权益/行业主题]产品，那确实在牛市时涨幅可能不如它们，
但在震荡市和下跌市中，本产品的[回撤控制/稳定收益]优势会更明显。建议您用『同样风险水平下的收益』
来比较，而不是单纯比收益率。」
```

## 四、波动期安抚话术

### 轻度波动（周跌幅 < 2%）

```
「本周市场正常调整，组合波动在预期范围内。债券端仍稳定贡献收益，建议保持耐心。」
```

### 中度波动（周跌幅 2-5%）

```
「本周市场受[具体因素]影响出现较大幅度调整，我能理解您的担忧。但从历史数据看，
这类级别的波动本产品平均在[X]天内修复。债券和黄金配置起到了明显的缓冲作用，
组合整体跌幅小于[同类平均/基准]。建议您先不要急于操作，给市场一点时间。」
```

### 重度波动（周跌幅 > 5%）

```
「本周市场出现了较为极端的波动，我第一时间联系您，就是想和您同步情况、稳定情绪。
从归因分析看，本次下跌主要由[具体因素]驱动，属于[系统性风险/行业集中释放]，
而非产品本身策略失效。我们的[再平衡/风控]机制会在必要时启动。当前最重要的是避免情绪化决策，
『卖在底部』比『买在顶部』对长期收益的损害更大。建议您[具体建议]，我会持续关注并及时沟通。」
```
