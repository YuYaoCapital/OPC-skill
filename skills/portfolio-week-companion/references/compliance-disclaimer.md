## 三、禁止用语清单

以下词汇/表述**绝对禁止**出现在面向客户的陪伴报告中：

| 禁止类别 | 具体词汇 |
|----------|----------|
| 收益承诺 | 保本、保收益、稳赚、肯定赚钱、必然上涨、绝对安全、零风险 |
| 绝对化表述 | 肯定、必然、一定、绝对、毫无疑问、板上钉钉 |
| 夸大宣传 | 最好、最强、顶级、王牌、神基、暴利、躺赚 |
| 误导性比较 | 跑赢大盘XX%（未注明时间区间和统计方法） |
| 不当暗示 | 内幕消息、机构护盘、国家队入场、必涨 |
| 紧迫性诱导 | 最后机会、错过再等一年、限时抢购、名额有限 |

## 四、谨慎用语清单

以下词汇/表述可以使用，但**须谨慎使用**，并配合充分的解释说明：

| 谨慎用语 | 使用条件 |
|----------|----------|
| "估值较低" | 须注明估值分位（如"处于近5年35%分位"） |
| "配置价值" | 须注明是"中长期"而非"短期" |
| "值得关注" | 不能暗示"立即买入" |
| "有望" / "可能" | 须配合不确定性说明 |
| "建议持有" | 须说明理由，不能无理由建议 |
| "分批建仓" | 须说明适用场景和节奏 |

## 五、基金经理观点引用规范

1. **必须注明来源**："数据来源：XX基金XXXX年X季报 | 基金经理：XXX"
2. **不得断章取义**：引用段落须保持原意完整，不得截取只言片语误导客户
3. **不得过度解读**：基金经理的观点仅反映季报时点判断，不代表未来持续看好/看空
4. **季报过期处理**：若最新季报已发布超过3个月，须注明"以上观点来自X季度报告，仅供参考"
