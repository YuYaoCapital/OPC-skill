**浏览器端指数获取代码**：
```javascript
// P4 东方财富 — 支持 CORS，可直接 fetch
async function fetchIndexQuote(secid) {
  const url = `https://push2.eastmoney.com/api/qt/stock/get?secid=${secid}&fields=f43,f44,f45,f46,f57,f58,f60,f170,f171`;
  const res = await fetch(url);
  const d = (await res.json()).data;
  return {
    price: (d.f43 / 100).toFixed(2),
    changePct: (d.f171 / 100).toFixed(2)
  };
}

// 常用 secid 映射
const SECID_MAP = {
  'hs300': '1.000300', 'sse': '1.000001', 'chi_next': '0.399006'
};
```

**降级策略（恒生指数示例）**：
```javascript
const hsi = await fetchWithFallback([
  () => fetchIndexQuote('116.HSI'),           // P4 东财
  () => fetchYahooViaProxy('^HSI'),            // P4 Yahoo（需代理）
  () => fetchFromCache('hsi')                 // P9 本地缓存
]);
```

## 数据缺失时的降级策略（浏览器端）

| 缺失数据 | 降级方案 |
|----------|----------|
| 底层基金实时净值 | 显示"数据更新中"，或隐藏该模块；优先用天天基金 JSONP |
| 美股数据 | 通过 Cloudflare Worker 代理 `/api/yahoo` 获取；如不可用显示"--" |
| 恒生指数 | 东财 `116.HSI` → Yahoo `^HSI`（代理）→ 本地缓存 → 显示"--" |
| Brinson归因 | 改为定性描述（"权益端小幅拖累"） |
| 基金经理季报 | 显示"季报尚未发布"，或引用上期观点并注明 |
| 历史回撤数据 | 隐藏模块9，或用"成立以来最大回撤"替代 |
| 投研观点未到位 | Xueqi 基于公开信息撰写，注明"基于公开资料" |

**浏览器端通用降级函数**：
```javascript
async function fetchWithFallback(fetchers) {
  for (const fetcher of fetchers) {
    try {
      const data = await fetcher();
      if (data && data.price !== undefined) return data;
    } catch (e) { console.warn('Source failed:', e.message); }
  }
  return null;
}
```
