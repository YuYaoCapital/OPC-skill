# 基金电话营销合规要求

## 一、监管法规依据

- 《证券投资基金销售管理办法》
- 《证券期货投资者适当性管理办法》
- 《关于规范金融机构资产管理业务的指导意见》（资管新规）
- 《基金募集机构投资者适当性管理实施指引》
- 《证券基金经营机构董事、监事、高级管理人员及从业人员监督管理办法》

## 二、禁止行为清单

### 绝对禁止的词汇
以下词汇严禁在电话营销中使用：

| 禁止词汇 | 合规替代用语 |
|---------|------------|
| 保本 | 本金不受保障，投资有风险 |
| 保收益 | 收益不确定，过往业绩不代表未来 |
| 稳赚 | 存在亏损可能 |
| 无风险 | 有风险，风险等级为[X] |
| 高收益 | 预期收益[X]%（仅业绩比较基准） |
| 肯定涨 | 市场有波动，无法预测 |
| 必赚 | 投资有风险，可能亏损 |
| 安全 | 有风险，不保证本金安全 |
| 存款替代品 | 不同于存款，有风险 |
| 政府背书/国家支持 | 客观描述投资方向，不暗示政府担保 |

### 禁止行为
1. **虚假宣传** - 不得夸大产品收益或隐瞒风险
2. **诋毁同行** - 不得贬低其他机构或产品
3. **高压销售** - 不得制造紧迫感强迫客户决策
4. **代客操作** - 不得代替客户进行风险测评或签署文件
5. **承诺收益** - 不得以任何方式承诺保本保收益
6. **误导陈述** - 不得将基金与存款、国债等产品混淆
7. **违规推介** - 不得向风险承受能力不匹配的客户推介产品

## 三、必须履行的义务

### 1. 风险揭示义务
每次推介基金产品时必须包含以下内容：

> **标准话术:**
> "本基金有风险，投资需谨慎。基金过往业绩不代表未来表现，基金管理人管理的其他基金业绩也不构成对本基金业绩表现的保证。请您仔细阅读基金合同和招募说明书，了解基金的具体风险。"

### 2. 适当性匹配义务
必须确认客户风险承受能力与产品风险等级匹配：

| 客户类型 | 可购买产品风险等级 |
|---------|------------------|
| 保守型 (C1) | 低风险 (R1) |
| 稳健型 (C2) | 低、中低风险 (R1-R2) |
| 平衡型 (C3) | 低、中低、中等风险 (R1-R3) |
| 成长型 (C4) | 低、中低、中等、中高风险 (R1-R4) |
| 进取型 (C5) | 全部风险等级 (R1-R5) |

**话术模板:**
> "根据您之前的测评，您的风险承受能力是[类型]，适合购买[风险等级]及以下的产品。这只基金的风险等级是[等级]，与您的风险承受能力匹配。"

### 3. 信息披露义务
必须向客户披露以下信息：
- 基金名称、代码、类型
- 基金管理人、基金经理
- 风险等级
- 费率结构（申购费、管理费、托管费、赎回费）
- 业绩比较基准
- 投资方向和范围

### 4. 录音留痕义务
- 电话营销必须全程录音
- 录音保存期限不少于20年
- 客户同意后方可录音（需明确告知）

**话术模板:**
> "本次通话将会录音，用于服务品质监督，请问您是否同意？"

## 四、电话营销流程合规要点

### 第一步: 身份声明
> "您好，我是[公司名称]的理财经理[姓名]，工号[工号]。"

### 第二步: 获得沟通许可
> "今天致电是想和您分享一些投资理财信息，请问您现在方便接听电话吗？"

⚠️ **注意**: 如客户表示不方便，必须礼貌结束通话，约定下次联系时间。

### 第三步: 风险承受能力确认
> "根据我们的记录，您上次测评的风险承受能力是[类型]，请问您的情况有变化吗？"

⚠️ **注意**: 如客户风险承受能力发生变化，需重新进行测评。

### 第四步: 产品介绍与风险揭示
必须包含:
1. 产品基本信息
2. 风险等级说明
3. 风险揭示声明
4. 费用说明

### 第五步: 客户确认
> "请问您是否理解这只基金的风险特征？是否确认要申购？"

### 第六步: 后续服务说明
> "投资后我会持续跟进产品运作情况，定期向您汇报。如有任何问题，您可以随时联系我。"

## 五、特定场景合规话术

### 收益说明
❌ 错误: "这个基金年化收益可以达到15%"
✅ 正确: "这只基金过去3年的年化收益率是15%，但过往业绩不代表未来表现，实际收益可能高于或低于这个数字"

❌ 错误: "买这个肯定比存款划算"
✅ 正确: "基金不同于存款，收益不确定，也有可能亏损。从历史数据看，长期持有的收益率通常高于存款，但不保证"

### 风险说明
❌ 错误: "这个基金很安全"
✅ 正确: "这只基金的风险等级是中等，投资于股票市场，短期内可能出现10%-20%的波动"

❌ 错误: "放心吧，不会亏的"
✅ 正确: "基金投资有风险，可能会亏损。但从长期看，如果选对了产品和时机，获得正收益的概率较高"

### 费用说明
❌ 错误: "没有手续费"
✅ 正确: "申购费率是1.5%，现在打4折后是0.6%。另外每年还有0.15%的托管费，请确认您了解这些费用"

### 比较说明
❌ 错误: "我们的产品比XX银行的好多了"
✅ 正确: "不同平台的产品各有特点，我们的优势在于[具体优势，如费率、服务等]，您可以根据自己的需求选择"

## 六、客户投诉处理合规要点

### 投诉受理
- 必须耐心倾听，不得推诿
- 记录投诉内容和客户诉求
- 告知客户处理流程和时限

**话术模板:**
> "感谢您反馈这个问题，我会认真记录并上报。我们将在[X]个工作日内给您答复，请您放心。"

### 投诉处理
- 及时调查核实
- 与客户沟通解决方案
- 如属公司责任，依法赔偿
- 如不属公司责任，耐心解释

### 不得行为
- 不得威胁、恐吓客户
- 不得泄露客户投诉信息
- 不得故意拖延处理
- 不得打击报复客户

## 七、合规检查清单

每次电话营销前，确认以下事项：

- [ ] 已取得客户同意进行电话营销
- [ ] 已告知客户通话将录音
- [ ] 已确认客户风险承受能力
- [ ] 所推介产品与客户风险等级匹配
- [ ] 已准备完整的产品资料
- [ ] 熟悉产品的风险特征和费用结构
- [ ] 未使用任何禁止词汇
- [ ] 已准备风险揭示话术
- [ ] 已准备适当性匹配话术

## 八、违规后果

### 对机构的处罚
- 警告、罚款
- 暂停或取消销售资格
- 责令整改
- 公开谴责

### 对个人的处罚
- 警告、罚款
- 暂停或取消从业资格
- 记入诚信档案
- 追究法律责任（构成犯罪的）

## 九、合规话术示例

### 完整合规开场白
> "您好，请问是[客户姓氏]先生/女士吗？我是[公司名称]的理财经理[姓名]，工号[工号]。今天致电是想向您介绍一款基金产品。本次通话将录音，用于服务质量监督，请问您是否同意？另外，请问您现在方便接听吗？"

### 完整合规产品介绍
> "我给您介绍的是[基金名称]（代码：[代码]），这是一只[基金类型]基金，风险等级为[R]。基金经理是[姓名]，从业[年限]年。这只基金投资于[投资方向]，业绩比较基准是[基准]。
>
> 需要提醒您的是，基金投资有风险，过往业绩不代表未来表现，不保证本金和收益。根据您的风险测评，您的风险承受能力是[类型]，与这只基金的风险等级匹配。
>
> 基金的费率包括：申购费[费率]（目前[X]折后为[X]%）、管理费[费率]%、托管费[费率]%、赎回费根据持有时间从0到[X]%不等。请问您是否了解这些费用？"

### 完整合规促成话术
> "如果您确认要申购，需要通过[渠道]操作。申购前请仔细阅读基金合同和招募说明书。投资后我会持续跟进，定期向您汇报产品情况。请问您是否确认申购[金额]元的[基金名称]？"

## 十、特殊情况处理

### 客户要求承诺收益
> "非常抱歉，根据监管规定，我不能向您承诺任何收益。基金投资有风险，收益是不确定的。我能告诉您的是历史业绩和风险特征，供您参考。"

### 客户要求保本
> "根据资管新规，所有理财产品都不能承诺保本保收益。基金投资有可能亏损本金，这是您需要了解的风险。如果您完全不能承受本金损失，建议您考虑存款或国债等产品。"

### 客户风险不匹配仍要购买
> "根据适当性管理要求，我只能向您推介与您风险承受能力匹配的产品。这只基金的风险等级高于您的承受能力，我不能向您推介。我可以推荐几只与您风险等级匹配的产品，您看可以吗？"
