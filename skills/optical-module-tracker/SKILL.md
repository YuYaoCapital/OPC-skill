# 光模块产业链跟踪工具

## 快速定位

- **投资研究框架与估值方法**：见 [references/investment_framework.md](references/investment_framework.md)

## 工作模式

### 模式A：问答咨询
用户提出具体问题，提供结构化、精准的产业链知识。

**常见场景：**
- "光模块产业链有哪些环节？"
- "800G光模块的国产厂商有哪些？"
- "硅光和CPO有什么区别？"
- "中际旭创和新易盛怎么对比？"
- "光模块上游光芯片的瓶颈在哪？"

**执行流程：**
1. 判断问题属于产业链/公司/投资/技术哪个维度
2. 加载对应参考文件获取详细信息
3. 直接给出结构化回答，引用具体公司和数据

### 模式B：跟踪/分析报告生成
用户需要生成系统性的产业链跟踪或分析报告。

**常见场景：**
- "生成一份光模块产业链跟踪报告"
- "帮我写个光模块行业月报"
- "分析光模块当前景气度"
- "对比几家光模块龙头公司"

**执行流程：**
1. 明确报告主题、时间范围和侧重维度
2. 加载两个参考文件获取框架知识
3. 按需搜索最新行业动态和公司信息
4. 按以下结构生成报告：


## 输出规范

### 图文并茂要求
- **产业链图谱**：用 ASCII/文字流程图或 Markdown 表格展示产业链结构
- **对比表格**：公司对比、技术路线对比用结构化表格
- **数据可视化**：用 Markdown 表格呈现关键数据趋势
- **关键加粗**：核心结论、重要数据、重点公司名称加粗

### 数据准确性
- 市场规模、份额、增速等数据标注来源或时间
- 不确信的数据标注"据公开资料"
- 公司信息以公开披露为准

### 维度覆盖检查清单
生成报告时，根据用户需求勾选覆盖维度：

- [ ] 产业链全景（上中下游）
- [ ] 核心公司分析
- [ ] 投资研究（景气度/估值/业绩）
- [ ] 技术路线（硅光/CPO/LPO/速率演进）
- [ ] 最新行业动态
- [ ] 风险因素

## 核心知识速查

### 产业链成本结构
- 光器件：占光模块成本 74%
  - TOSA/ROSA：占光器件成本 80%
    - 光芯片：占 TOSA/ROSA 成本 85%
- 光芯片整体：占光模块成本约 26%

### 关键数据速记
- 全球市场规模：2024年178亿美元 → 2029年415亿美元（CAGR 18%）
- 全球前十中中国占7席
- 800G CAGR(20-24)：188%；1.6T CAGR(24-29)：180%
- 产线投资：800G约5亿元，1.6T约6亿元

### 国产替代进度
- 2.5G光芯片：>90%
- 10G光芯片：~60%
- 25G+光芯片：较低，突破中
- 光器件：较高
- 光模块：全球领先
- 设备：快速提升中
