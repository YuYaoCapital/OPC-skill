## 业绩跟踪框架

### 核心财务指标

**收入端：**
- 总收入增速（季度同比/环比）
- 分产品收入（100G/400G/800G/1.6T占比）
- 分客户收入（前五大客户集中度）
- 分地区收入（海外/国内占比）

**利润端：**
- 毛利率（产品结构、规模效应、价格竞争）
- 净利率
- 单季度净利润增速

**关键观察：**
- 新品放量时毛利率往往先降后升（爬坡期良率问题）
- 规模效应显著，龙头毛利率通常高于二线5-10pct

### 运营指标

| 指标 | 计算公式 | 关注要点 |
|------|---------|---------|
| 存货周转天数 | 平均存货/营业成本×365 | 需求旺盛时下降，过剩时上升 |
| 应收账款周转天数 | 平均应收/营业收入×365 | 大客户账期较长 |
| 产能利用率 | 实际产出/设计产能 | 高利用率支撑毛利 |
| 研发费用率 | 研发费用/收入 | 高端产品需持续投入 |

### 公司对比维度

1. **产品矩阵**：覆盖速率段是否完整？高端产品占比？
2. **客户结构**：绑定头部云厂商？客户集中度风险？
3. **垂直整合**：是否自产光芯片/光器件？成本优势？
4. **产能布局**：国内+海外双基地？应对关税/溯源？
5. **技术储备**：硅光/CPO/LPO布局进度？

## 估值方法

### 适用估值方法

| 方法 | 适用场景 | 公式/要点 |
|------|---------|----------|
| PE估值 | 成熟期、盈利稳定 | 股价/EPS；参考历史中枢 |
| PEG估值 | 高成长阶段 | PE/净利润增速；合理区间0.8-1.5 |
| PS估值 | 早期或盈利波动大 | 市值/收入；适用于新品放量期 |
| EV/EBITDA | 跨市场比较 | 企业价值/息税折旧前利润 |
| DCF | 长期价值评估 | 需假设长期增速和终值 |

### 估值参考锚点

- **行业历史PE区间**：通常20x-40x（景气上行可到50x+，下行可到15x）
- **龙头溢价**：中际旭创等全球龙头通常享有10-20%估值溢价
- **业绩确定性溢价**：订单可见度高、客户绑定深的公司溢价

### 估值驱动因素

**向上驱动：**
- 云厂商Capex超预期
- 新品（800G/1.6T）放量加速
- 毛利率因规模效应提升
- 国产替代突破（光芯片）

**向下驱动：**
- 云厂商Capex下修
- 价格竞争加剧
- 技术路线被颠覆（如CPO替代可插拔）
- 地缘政治/关税风险

## 核心风险因素

### 需求风险
- **AI投资周期性**：若AI需求不及预期，云厂商削减Capex
- **技术迭代风险**：新速率产品推出慢于预期
- **客户集中风险**：头部客户订单波动影响大

### 供给风险
- **光芯片短缺**：高端光芯片仍依赖海外
- **价格竞争**：国内厂商扩产快，可能引发价格战
- **产能过剩**：行业周期下行时库存积压

### 地缘政治风险
- **关税**：美国对中国光模块加征关税
- **溯源要求**：要求证明非中国制造/非中国芯片
- **实体清单**：关键企业被制裁

### 技术路线风险
- **CPO替代**：若CPO快速普及，传统光模块价值量下降
- **硅光渗透率**：硅光改变产业格局，新进入者涌现
- **铜缆替代**：短距场景铜缆（DAC）替代光模块

## 周期性特征

光模块行业具有**明显的周期性**，周期长度约3-5年：

```
上行周期：新技术导入 → 云厂商扩产 → 订单爆发 → 产能扩张 → 业绩释放
下行周期：需求饱和 → 库存积压 → 价格下滑 → 毛利率压缩 → 产能过剩
```

**当前周期位置判断要点**：
- AI算力投资是本轮周期的核心驱动力
- 800G/1.6T迭代速度快于历史周期
- 需关注云厂商Capex指引变化

**投资时钟**：
- 周期底部：关注估值修复+订单拐点信号
- 周期上行：关注业绩兑现+产能释放
- 周期顶部：警惕毛利率拐点+库存风险
- 周期下行：防御为主，等待下一轮技术迭代
