# 基金市场波动客户维护话术 Skill

## 概述

本 Skill 用于帮助理财经理在以下场景进行客户维护：
- 基金净值出现回撤，客户情绪焦虑
- 市场波动加剧，需要主动进行客户安抚
- 定期客户回访，汇报基金运作情况
- 引导客户正确看待短期波动，避免非理性赎回

支持基金类型：
- **固收+基金**（债券为主+权益增强）：如二级债基、偏债混合、灵活配置型
- **权益基金**（偏股混合型）：如产业臻选、趋势优先等主动管理型产品

## 输入参数

使用本 Skill 时，请提供以下信息：

| 参数 | 说明 | 示例 |
|------|------|------|
| `基金代码` | 基金的唯一标识代码 | 519768、025002 |
| `基金类型` | 基金的产品类型 | 固收+、偏股混合、二级债基 |
| `数据日期` | 最新净值数据的日期 | 2026年4月8日 |
| `客户类型` | 客户分类（可选，默认生成全部8类） | 主动购买/正收益/超配 |

## 输出内容结构

生成的维护话术文档包含以下模块：

### 1. 基金基本信息
- 基金名称、代码、最新净值
- 业绩表现（近1月/3月/6月/1年、成立以来）
- 风险指标（最大回撤、股票仓位等）
- 基金经理简介及最新观点

### 2. 重仓股详细分析（如适用）
- 前十大重仓股列表及持仓占比
- 各重仓股近3个月涨跌幅
- **基本面情况**：公司业务、财务状况
- **热点事件解析**：近期重大新闻、业绩公告、政策影响
- **对基金的影响**：个股波动对基金净值的贡献度

### 3. 市场环境分析
- 债券市场：收益率走势、货币政策
- 股票市场：结构性分化、板块轮动
- 宏观经济：政策导向、外部冲击

### 4. 八类客户分类话术

根据客户特征矩阵生成差异化话术：

| 维度1 | 维度2 | 维度3 | 客户类型 | 标注颜色 |
|-------|-------|-------|----------|----------|
| 主动购买 | 正收益 | 超配 | 类型一 | 🔵 蓝色 |
| 主动购买 | 负收益 | 超配 | 类型二 | 🟠 橙色 |
| 主动购买 | 正收益 | 非超配 | 类型三 | 🟢 绿色 |
| 主动购买 | 负收益 | 非超配 | 类型四 | 🔴 红色 |
| 被动购买 | 正收益 | 超配 | 类型五 | 🟣 紫色 |
| 被动购买 | 负收益 | 超配 | 类型六 | 🔴 深红色 |
| 被动购买 | 正收益 | 非超配 | 类型七 | 🩵 青色 |
| 被动购买 | 负收益 | 非超配 | 类型八 | 🟤 棕色 |

每类客户话术包含：
- **沟通重点**：该类客户的核心沟通策略
- **开场白话术**：致电客户的开场表述
- **情况分支话术**：根据客户不同反应的应对话术
- **调仓建议**：具体的操作建议

### 5. 调仓及赎回对接策略
- 超配客户的减仓建议
- 亏损客户的持有/赎回决策框架
- 赎回资金对接方案（短债基金、货币基金等）

### 6. 产品要点速查
- 理财经理备查的核心信息
- 核心话术要点总结
- 风险提示事项

## 使用示例

### 示例一：固收+基金（519768 交银优选回报）

**输入参数：**
```
基金代码：519768
基金类型：固收+（灵活配置型）
数据日期：2026年4月8日
```

**输出内容特点：**
- 突出债券底仓的安全性（85%+债券配置）
- 强调高股息策略的稳健性
- 对比纯债基金和股票基金的风险收益特征
- 话术风格：安抚、稳健、强调长期持有

### 示例二：偏股混合基金（025002 交银产业臻选）

**输入参数：**
```
基金代码：025002
基金类型：偏股混合型
数据日期：2026年4月8日
```

**输出内容特点：**
- 详细分析重仓股（泡泡玛特、游戏股、CXO等）
- 解读个股暴跌对基金的影响（如泡泡玛特拖累净值1.8-2%）
- 基金经理投资策略分析
- 话术风格：解释回撤原因、安抚情绪、提供解决方案

## 执行流程

1. **确认基金代码**：获取准确的基金代码和名称
2. **收集基础数据**：
   - 从天天基金网获取最新净值、业绩数据
   - 从基金年报/季报获取重仓股信息
   - 从天天基金网、东方财富网获取重仓股基本面和热点新闻
3. **市场环境分析**：
   - 搜索最新债市/股市行情分析
   - 整理基金经理最新观点
4. **生成分类话术**：
   - 根据八类客户矩阵生成差异化话术
   - 针对基金类型调整话术风格（固收+偏稳健，权益偏解释）
5. **输出文档**：
   - Word版本：可编辑、便于打印
   - PDF版本：带彩色标注（不同客户类型用不同颜色区分）

## 话术设计原则

### 1. 固收+基金话术要点
- **强调安全性**：债券底仓提供安全垫
- **突出稳健性**：对比股票基金的低波动性
- **传递信心**：短期波动正常，耐心持有等待修复
- **配置建议**：择机降低超配部分，均衡配置

### 2. 权益基金话术要点
- **解释回撤原因**：具体分析重仓股波动影响
- **安抚情绪**：承认波动，但强调基本面未变
- **提供方案**：持有/定投/减仓的选择框架
- **风险提示**：偏股基金波动大，需控制仓位

### 3. 不同客户类型沟通策略

| 客户类型 | 沟通策略 | 核心话术 |
|----------|----------|----------|
| 正收益+超配 | 提醒风险，建议减仓 | "虽然盈利，但仓位偏高，建议降低风险敞口" |
| 负收益+超配 | 安抚情绪，建议减仓+持有 | "理解您的心情，建议先降低仓位，保留部分等待反弹" |
| 正收益+非超配 | 确认配置，建议持有 | "配置合理，建议继续持有，但密切关注波动" |
| 负收益+非超配 | 安抚情绪，建议耐心持有 | "配置合理，浮亏是短期的，建议耐心等修复" |
| 被动购买+亏损 | 表达歉意，提供补救 | "为推荐时点不佳致歉，建议调整仓位，我们会持续跟踪" |

## 注意事项

1. **数据准确性**
   - 所有数据需来自天天基金网、东方财富网等权威来源
   - 标注数据更新日期，确保时效性
   - 重仓股数据以最新季报为准

2. **合规要求**
   - 话术仅供理财经理参考，不构成投资建议
   - 需提醒客户"基金有风险，投资需谨慎"
   - 过往业绩不预示未来表现

3. **客户情绪管理**
   - 对亏损客户先安抚情绪，再解释原因
   - 对被动购买的亏损客户需表达适当歉意
   - 避免使用"肯定回本"等绝对化表述

4. **文档输出**
   - Word版本便于编辑和打印
   - PDF版本带彩色标注，不同客户类型用不同颜色区分，便于快速查找
