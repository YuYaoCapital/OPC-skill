# 固收+基金跟踪指标详解

## 一、监控阈值参考值

### 1. 净值波动阈值

| 基金类型 | 单日跌幅预警 | 单日跌幅关注 | 连续下跌触发 |
|---------|-------------|-------------|-------------|
| 稳健型固收+（股票仓位<15%） | -0.30% | -0.50% | 连续3日累计<-0.80% |
| 均衡型固收+（股票仓位15-25%） | -0.50% | -0.80% | 连续3日累计<-1.50% |
| 进取型固收+（股票仓位>25%） | -0.80% | -1.20% | 连续3日累计<-2.00% |

### 2. 回撤阈值

| 阶段 | 历史最大回撤占比 | 对应级别 |
|------|-----------------|---------|
| 正常 | < 30% | 绿色 |
| 关注 | 30% - 50% | 黄色 |
| 预警 | 50% - 70% | 橙色 |
| 危险 | 70% - 100% | 红色 |
| 突破历史 | > 100% | 深红（触发重评） |

### 3. 同类排名阈值

| 状态 | 同类排名分位 | 说明 |
|------|-------------|------|
| 优秀 | 前20% | 无需关注 |
| 良好 | 20%-40% | 正常跟踪 |
| 一般 | 40%-60% | 观察原因 |
| 落后 | 60%-80% | 纳入周报分析 |
| 尾部 | 后20% | 深度排查 |

## 二、关键指标计算方法

### 当前回撤
```
当前回撤 = (当前净值 - 历史最高净值) / 历史最高净值 × 100%
```

### 回撤修复进度
```
修复进度 = (当前净值 - 回撤最低点) / (历史最高净值 - 回撤最低点) × 100%
```

### 滚动收益
```
近N日收益 = (当前净值 / N日前净值 - 1) × 100%
```

### 行业集中度 HHI
```
HHI = Σ(各行业配置比例²) × 10000
HHI变化 = 本期HHI - 上期HHI
（变化>100视为显著变动）
```

## 三、市场环境关联指标

| 市场因素 | 观察指标 | 对固收+基金影响 |
|---------|---------|----------------|
| 利率债 | 10Y国债收益率变动 | 反向：收益率↑，债券价格↓ |
| 信用债 | 信用利差（AAA3Y-国债3Y） | 利差↑，信用债承压 |
| 股市 | 沪深300涨跌幅 | 正向：影响股票仓位收益 |
| 可转债 | 中证转债指数 | 正向：影响可转债仓位 |
| 流动性 | 银行间质押回购利率(DR007) | 资金紧→债基承压 |

## 四、跟踪检查清单

### 每日检查
- [ ] 净值是否更新
- [ ] 单日涨跌幅是否触发预警
- [ ] 是否连续下跌

### 每周检查
- [ ] 周收益及同类排名
- [ ] 当前回撤状态
- [ ] 相对基准超额变化
- [ ] 市场环境总结

### 每月检查
- [ ] 月报生成
- [ ] 滚动收益分析（1月/3月/YTD）
- [ ] 同类对比（3-5只）
- [ ] 规模变动

### 季报节点检查
- [ ] 持仓对比分析
- [ ] 久期/信用结构变化
- [ ] 股票仓位/重仓股变化
- [ ] 基金经理观点更新
