# 微博热点财经情报板

## 用途

将每日微博热搜财经话题、核心投资热点、风险警示、机构动向、投资观点等数据，
渲染为一张专业排版的情报板长图，适合公众号、社群、投研分享。

## 触发场景

- "帮我生成今日财经热点图"
- "做一张微博热搜情报板"
- "把今天的财经新闻做成图"
- "每日财经可视化"
- 用户提供了一组热点数据，要求"做成图"

## 工作流程

### 第一步：获取数据

从以下任一来源获取当日财经热点数据：

1. **用户提供**：用户直接复制粘贴新闻内容
2. **网页抓取**：如用户提供了微博/财经网站链接，用工具抓取内容
3. **手动整理**：根据已有信息结构化整理

字段说明：
- `color` 可选值：`red` / `orange` / `blue` / `purple` / `green`
- `hot_topics`：微博热搜榜单，显示排名、话题、热度
- `hotspots`：核心投资热点，2列网格展示
- `risks`：风险警示列表，独立红色卡片
- `institutions`：机构动向列表，独立绿色卡片
- `views`：投资观点速查，3列药丸标签

### 第三步：运行生成脚本

将 JSON 保存到文件后，执行：

```bash
python3 ~/.kimi/skills/weibo-finance-daily/scripts/generate_board.py \
  --data data.json \
  --output /path/to/output.png \
  --width 1000
```

参数说明：
- `--data`: JSON 数据文件路径（必填）
- `--output`: 输出图片路径，默认 `board.png`
- `--width`: 画布宽度，默认 1000px

### 第四步：交付图片

将生成的 PNG 图片路径告知用户。

## 视觉规范

| 元素 | 规范 |
|------|------|
| 背景 | 淡黄色 `#FEF9E7` |
| 卡片 | 纯白 `#FFFFFF`，圆角 12px，细边框 |
| 字体 | Hiragino Sans GB Bold（或系统黑体 fallback），全篇加粗 |
| 大标题 | 52px，居中 |
| 模块标题 | 26px，彩色色块衬底 |
| 条目标题 | 22px，最重要阅读层 |
| 数据标签 | 24px，右对齐 |
| 正文 | 16px，辅助文字 #64748B |
| 风险卡片 | 独立浅红底 + 左侧红条 |
| 机构卡片 | 独立浅绿底 + 左侧绿条 |
| 观点标签 | 药丸形，彩色填充 |

## 依赖

- Python 3
- Pillow (`pip install Pillow`)
- 系统中文字体（Hiragino Sans GB / STHeiti / PingFang）

## 注意事项

1. 若系统无 Hiragino Sans GB，脚本会自动降级到 STHeiti 等可用字体
2. 所有字段均为可选，缺少的模块会自动跳过不渲染
3. 画布高度自适应内容，宽度固定 1000px
