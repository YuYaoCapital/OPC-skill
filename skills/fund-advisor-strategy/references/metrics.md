# 基金评价指标体系

## 收益类指标

### 收益率指标

| 指标 | 计算公式 | 评价标准 |
|------|----------|----------|
| 年化收益率 | $(1+总收益率)^{(365/天数)}-1$ | 同类前1/3为优秀 |
| 超额收益(α) | 基金收益 - 基准收益 | >0 为跑赢基准 |
| 信息比率 | 超额收益 / 跟踪误差 | >0.5 较好，>1.0 优秀 |

### 风险类指标

| 指标 | 计算公式 | 评价标准 |
|------|----------|----------|
| 波动率(σ) | 收益率标准差(年化) | 越低越好 |
| 最大回撤 | $\frac{峰值-谷底}{峰值}$ | 同类前1/3为优秀 |
| 下行标准差 | 负收益的标准差 | 衡量下行风险 |
| VaR(95%) | 5%分位数收益 | 95%置信度下的最大损失 |

### 风险调整后收益

| 指标 | 计算公式 | 评价标准 |
|------|----------|----------|
| 夏普比率 | $\frac{R_p - R_f}{\sigma_p}$ | >1.0 较好，>2.0 优秀 |
| 卡玛比率 | $\frac{年化收益率}{最大回撤}$ | >2.0 优秀 |
| 索提诺比率 | $\frac{R_p - R_f}{下行标准差}$ | 夏普比率的改进版 |
| 特雷诺比率 | $\frac{R_p - R_f}{\beta}$ | 单位系统风险的收益 |

## 基金筛选指标

### 基础筛选条件

**成立时间**: >3年（观察完整牛熊周期）
**基金规模**: 2亿-100亿（规模适中）
**基金经理**: 任职>2年（风格稳定）

### 多维度评分体系

**收益能力(25%)**
- 近1年同类排名: 权重10%
- 近3年同类排名: 权重15%

**风险控制(25%)**
- 最大回撤排名: 权重15%
- 波动率排名: 权重10%

**风险调整后收益(30%)**
- 夏普比率排名: 权重15%
- 卡玛比率排名: 权重15%

**稳定性(10%)**
- 基金经理稳定性: 权重5%
- 规模稳定性: 权重5%

**其他(10%)**
- 费率水平: 权重5%
- 机构持有比例: 权重5%

## 组合分析指标

### 集中度指标

**个股集中度**
```
前十大重仓占比 = 前十大持仓市值 / 基金总净值
```
- >60% 为高度集中
- 40-60% 为中等集中
- <40% 为分散

**行业集中度**
```
赫芬达尔指数 = Σ(行业占比²)
```
- >0.3 为集中
- <0.15 为分散

### 相关性分析

**基金间相关系数**
```python
import pandas as pd

def calculate_correlation(fund_returns):
    """计算基金收益相关性矩阵"""
    return fund_returns.corr()
```

**组合分散度标准**
- 任意两只基金相关系数 < 0.8
- 权益类基金内部相关系数 < 0.9

### 久期与利率敏感度

**债券基金**
- 久期: 衡量利率敏感度
- 修正久期 × 利率变化 ≈ 价格变化
- 信用债占比: 评估信用风险
