# 投资策略方法论

## 战略资产配置(SAA)

### 客户风险等级分类

| 风险等级 | 权益类上限 | 权益类下限 | 适合人群 |
|----------|------------|------------|----------|
| 保守型(C1) | 20% | 0% | 退休、保本为主 |
| 稳健型(C2) | 40% | 10% | 临近退休、低风险偏好 |
| 平衡型(C3) | 60% | 30% | 中年、中等风险承受 |
| 成长型(C4) | 80% | 50% | 年轻、有一定经验 |
| 进取型(C5) | 100% | 70% | 专业投资者 |

### 常见SAA模型

**60/40经典配置**
```
股票型基金: 60%
债券型基金: 40%
```

**年龄法则**
```
权益类比例 = 100 - 年龄
债券类比例 = 年龄
```

**目标日期配置(下滑轨道)**
```python
def glide_path(years_to_target):
    """距目标年限的权益类比例"""
    if years_to_target >= 25:
        return 0.90
    elif years_to_target >= 15:
        return 0.75
    elif years_to_target >= 10:
        return 0.60
    elif years_to_target >= 5:
        return 0.40
    else:
        return 0.20
```

## 战术资产配置(TAA)

### 动态调整方法

**估值择时法**
- 股债性价比(ERP) > 历史80%分位 → 增配权益
- 股债性价比(ERP) < 历史20%分位 → 减配权益

**趋势跟踪法**
- 沪深300站上20月均线 → 增配A股
- 10年期国债收益率下行 → 增配长久期债基

**美林时钟法**
```
复苏期 → 股票 > 债券 > 现金 > 商品
过热期 → 商品 > 股票 > 现金 > 债券
滞胀期 → 现金 > 商品 > 债券 > 股票
衰退期 → 债券 > 现金 > 股票 > 商品
```

## 基金定投策略

### 定投优化策略

**智能定投(均线法)**
```python
def smart_investment_amount(base_amount, current_index, ma250):
    """
    根据当前指数与年线关系调整定投金额
    """
    ratio = current_index / ma250
    if ratio < 0.85:      # 大幅低于均线
        return base_amount * 2.0
    elif ratio < 0.95:    # 低于均线
        return base_amount * 1.5
    elif ratio < 1.05:    # 接近均线
        return base_amount * 1.0
    elif ratio < 1.15:    # 高于均线
        return base_amount * 0.8
    else:                 # 大幅高于均线
        return base_amount * 0.5
```

**估值定投法**
- PE < 30%分位 → 定投金额的200%
- PE 30-50%分位 → 定投金额的150%
- PE 50-70%分位 → 定投金额的100%
- PE > 70%分位 → 定投金额的50%或暂停

## 组合再平衡策略

### 触发机制

**阈值法**
- 单一资产偏离目标配置 > 5% → 触发再平衡
- 单一基金权重偏离 > 3% → 触发调整

**定期法**
- 每季度/每半年检查一次
- 每年末固定再平衡

**事件法**
- 客户资金变动
- 基金更换经理
- 市场大幅波动(如回撤>15%)

### 再平衡方法

**完全再平衡** → 恢复到目标配置
**部分再平衡** → 调整50%偏离度
**增量调整** → 新增资金按目标比例配置
