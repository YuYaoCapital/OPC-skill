### 1.1 评分维度与权重

| 一级维度 | 权重 | 二级指标 | 指标权重 |
|----------|------|----------|----------|
| 收益能力 | 30% | 近1年收益排名 | 10% |
| | | 近3年收益排名 | 12% |

### 1.2 总分计算

```
综合得分 = Σ(各维度得分 × 维度权重)
```

## 三、评级标准

| 综合得分 | 投资评级 | 配置建议 | 关注频率 |

| 90-100 | ★★★★★ 强烈推荐 | 核心配置（15%-20%） | 季度回顾 |

### 3.2 不同类型基金调整系数

考虑到不同类型基金的风险收益特征不同，应用以下调整：

| 基金类型 | 收益期望 | 风险容忍 | 调整说明 |
|----------|----------|----------|----------|
| 股票型 | 高 | 高 | 标准评分 |
| 偏股混合 | 较高 | 较高 | 标准评分 |
| 灵活配置 | 中等 | 中等 | 标准评分 |
| 偏债混合 | 中等 | 低 | 夏普权重+5% |
| 债券型 | 低 | 低 | 夏普权重+10% |
| QDII | 高 | 高 | 考虑汇率因素 |
| 指数型 | 市场平均 | 市场平均 | 主要跟踪误差 |

## 六、评分注意事项

1. **数据完整性**：评分依赖于完整的历史数据，新成立基金（<2年）评分仅供参考
2. **市场环境**：极端市场环境下的表现可能需要特殊考虑
3. **基金经理变更**：如近期有基金经理变更，评分有效性下降
4. **规模变化**：短期内规模大幅变化可能影响策略有效性
5. **定期更新**：建议每季度重新评分，跟踪变化趋势
