# 主动管理型基金投资研究报告

本 skill 为理财经理生成专业的主动管理型基金投资研究报告（Word格式），报告内容全面、数据详实、图表丰富，可用于内部研究参考或客户沟通材料。

## 触发条件

用户以下述方式请求时触发：
- "生成 {基金代码} 的投资报告"
- "分析一下 {基金代码} 值不值得买"
- "写一份 {基金名称} 的研究报告"
- "帮我研究一下这只基金 {基金代码}"

## 核心功能

1. **自动数据采集**：从东方财富网抓取基金基础数据、净值数据、持仓数据
2. **深度业绩分析**：多维度业绩归因、风险调整后收益计算
3. **持仓结构解析**：行业分布、市值风格、重仓股质量评估
4. **基金经理画像**：从业经历、管理规模、投资风格、能力圈识别
5. **可视化图表**：自动生成业绩走势、风险收益、行业分布等图表
6. **专业Word报告**：生成10页以上的结构化研究报告

## 快速使用

### 生成单只基金报告

```bash
cd /Users/r9/.kimi/skills/fund-active-research
python scripts/generate_report.py 519702
```

### 带参数生成

```bash
# 指定输出路径
python scripts/generate_report.py 519702 --output ~/reports/

# 指定时间范围（年）
python scripts/generate_report.py 519702 --years 3
```

### Python 调用

```python
from scripts.generate_report import FundResearchReport
from scripts.fund_eastmoney import FundDataFetcher

# 获取数据
fetcher = FundDataFetcher()
fund_data = fetcher.get_fund_data('519702')

# 生成报告
generator = FundResearchReport()
report_path = generator.generate('519702', output_dir='~/reports/')
```

## 报告内容结构

报告共11个章节，10页以上：

### 第一章 基金概览
- 基金基本信息（名称、代码、类型、成立日期）
- 管理规模及变化趋势
- 基金经理信息（姓名、任职日期、从业年限）

### 第二章 历史业绩回顾
- 各周期收益率（近1月/3月/6月/1年/2年/3年/5年/成立至今）
- 同类排名及分位
- 年度收益分布
- 业绩走势图（图表）

### 第三章 风险分析
- 风险指标（波动率、最大回撤、夏普比率、卡玛比率）
- 回撤修复时间
- 风险收益散点图（图表）

### 第四章 业绩归因
- Brinson归因（资产配置收益 + 个股选择收益）
- 收益来源拆解
- 择时能力评估

### 第五章 持仓分析
- 行业分布（图表）
- 市值风格分布（大盘/中盘/小盘）
- 前十大重仓股分析
- 持仓集中度变化

### 第六章 调仓特征
- 换手率分析
- 重仓股稳定性
- 调仓节奏与有效性

### 第七章 基金经理画像
- 从业经历与管理规模
- 投资风格识别（价值/成长/均衡）
- 能力圈分析（擅长行业）
- 管理多只基金的业绩一致性

### 第八章 同类对比
- 与同类基金业绩对比
- 风险调整后收益对比
- 规模与流动性对比

### 第九章 优势与风险
- 核心投资亮点（3-5条）
- 主要风险点（3-5条）
- 适合投资者类型

### 第十章 投资价值判断
- 综合评分（1-100分）
- 评级建议（强烈推荐/推荐/中性/谨慎）
- 配置建议（核心/卫星/观察）

### 第十一章 前瞻性观点
- 后市展望
- 策略有效性预期
- 关注要点提示

## 数据源说明

数据来自东方财富网（天天基金网）：
- 基金基础信息：fundf10.eastmoney.com
- 净值数据：fund.eastmoney.com
- 持仓数据：cninfo.com.cn（巨潮资讯网）

## 注意事项

1. **数据时效性**：持仓数据来自季报，有一定滞后性
2. **网络依赖**：需要联网获取东方财富数据
3. **生成时间**：完整报告生成约需 1-3 分钟
4. **图表说明**：报告中的图表为静态图片，可直接复制使用
