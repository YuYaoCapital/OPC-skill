# 基金组合调仓发车内容生成器

为自媒体/KOL提供专业且易传播的基金组合调仓内容，帮助快速生成高质量的"发车"文案。

## 触发条件

用户以下述方式请求时触发：
- "生成调仓发车内容"
- "帮我写个调仓文案"
- "组合发车图文"
- "{XX基金组合}调仓建议"
- "发车文案+{组合名称}"

## 核心功能

1. **调仓逻辑生成** - 基于市场环境生成调仓理由
2. **多维度调仓方案** - 资产配置/行业轮动/具体标的/仓位管理
3. **多种风格可选** - 专业严谨/通俗易懂/轻松幽默
4. **多格式输出** - 话术文案/图文内容/海报素材
5. **传播优化** - 适合社交媒体传播的结构和话术

## 使用流程

### 第一步：收集调仓信息

询问或确认以下信息：
- **组合名称**：如"稳稳的幸福组合"、"进取成长组合"
- **调仓方向**：加仓/减仓/调仓换股/维持观望
- **调仓维度**（可多选）：
  - 大类资产配置（股债配比调整）
  - 行业/风格轮动（板块切换）
  - 具体基金标的（调入调出）
  - 仓位管理（整体仓位水平）
- **内容风格**：专业严谨/通俗易懂/轻松幽默
- **输出格式**：文案/图文/海报/多格式

### 第二步：生成调仓逻辑

基于当前市场环境，生成调仓理由框架：

**市场环境分析维度**：
- 宏观经济（PMI、通胀、利率）
- 市场估值（主要指数PE/PB分位）
- 资金流向（北向资金、公募发行）
- 情绪指标（成交量、波动率）
- 政策动向（货币政策、行业政策）

**调仓逻辑模板**：
```
1. 现状诊断：当前组合面临的主要问题/机会
2. 核心逻辑：本次调仓的主要理由（1-3条）
3. 操作方案：具体调仓动作说明
4. 预期效果：调仓后组合特征变化
```

### 第四步：生成传播内容

根据选择的风格生成相应内容：

#### 风格A：专业严谨型

特点：
- 数据支撑充分
- 专业术语准确
- 逻辑严密清晰

## 输出格式

### 格式1：话术文案

```
🎯 {组合名称}调仓发车

【调仓背景】
[市场环境简述]

【核心逻辑】
1. [逻辑点1]
2. [逻辑点2]
3. [逻辑点3]

【调仓方案】
📊 资产配置：[调整说明]
🔄 行业轮动：[调整说明]
💼 标的调整：[调入/调出/调仓]
📈 仓位建议：[仓位水平]

【预期效果】
[调仓后组合特征]

⚠️ 风险提示：以上建议仅供参考，不构成投资建议。基金有风险，投资需谨慎。

### 格式2：图文内容

生成带排版标记的图文内容，适合公众号/小红书/雪球等平台：

- 标题（吸引点击）
- 封面图建议
- 正文分段（配合图片位置标记）
- 配图建议（图表类型/数据）
- 互动引导（留言/点赞/转发话术）

### 格式3：海报素材

生成海报文案结构：

- 主标题（大字突出）
- 副标题（补充说明）
- 核心观点（3个以内）
- 调仓要点（可视化呈现）
- 二维码/关注引导

## 内容优化技巧

### 标题撰写

高点击率标题公式：
1. **数字+痛点**："3个信号告诉你，该调仓了"
2. **悬念+利益**："这次调仓，可能会影响你下半年的收益"
3. **场景+行动**："市场震荡，{组合名}这样调仓更稳健"
4. **对比+反差**："别人割肉，我们加仓：{组合名}调仓思路"

### 开头钩子

3秒抓住注意力的开头方式：
- 数据冲击："最近XX指数下跌了XX%，但我们的组合..."
- 故事引入："有位粉丝问我，现在该不该..."
- 观点鲜明："直接说结论：现在要..."
- 悬念设置："市场发生了3个变化，很多人没注意到..."

### 互动设计

提升互动的技巧：
- 结尾提问："你现在的仓位是几成？评论区聊聊"
- 投票引导："看涨还是看跌？点个赞/留个1"
- 福利诱惑："转发此文，私信领取完整调仓清单"
- 悬念续集："明天分析另一个组合，记得关注"

2. 风格再平衡必要：成长/价值分化严重，需适度均衡

（表格形式呈现调仓详情...）

四、风险提示
...
```

先说结论：要动，但不用大动。

就像开车一样，前面路况有点变化，咱们微调一下方向就行，不用急转弯。

2️⃣ 科技板块加点仓
...
```

理由：债券最近有点"傲娇"，收益一般还占地方，先腾点空间

🎯 新增站点：科技成长站
...
```
