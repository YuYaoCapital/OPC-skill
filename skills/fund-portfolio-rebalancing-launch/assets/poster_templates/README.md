# 海报模板资源

## 说明

此目录用于存放海报设计参考模板和素材。

## 推荐海报设计工具

### 在线工具
- **Canva** (canva.com) - 推荐，有大量基金/金融模板
- **稿定设计** (gaoding.com) - 国内优秀设计工具
- **创客贴** (chuangkit.com) - 适合新媒体海报

### 设计要点

#### 配色方案

**专业严谨风格**：
- 主色：深蓝 #1e3a5f
- 辅色：金色 #c9a227
- 背景：白色/浅灰 #f5f5f5

**通俗易懂风格**：
- 主色：暖橙 #ff6b35
- 辅色：深蓝 #004e89
- 背景：米白 #faf9f6

**轻松幽默风格**：
- 主色：活力红 #e63946
- 辅色：明亮黄 #ffb703
- 背景：浅色渐变

#### 海报文案结构

```
【主标题】（最大字号）
🚗 {组合名}发车啦！

【副标题】
2025年Q2调仓方案

【核心观点】（3个以内）
✓ 权益仓位提升至50%
✓ 增配科技成长板块
✓ 调入2只优质基金

【二维码区域】
扫码关注，获取完整调仓清单

【风险提示】（底部小字）
以上内容仅供参考，不构成投资建议
```

## 注意事项

1. **合规要求**：海报上必须包含风险提示
2. **二维码**：确保可扫描，链接有效
3. **分辨率**：导出时选择高清
4. **文件大小**：控制在5MB以内
