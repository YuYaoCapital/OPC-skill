# R9Alpha 研基评价框架（偏股混合-主动）

## 框架说明

R9Alpha 研基是用于主动管理型基金深度评价的体系，包含十大维度：
1. 基础信息
2. 收益指标
3. 风险指标
4. 性价比指标
5. 超额指标
6. 持有人体验指标
7. 价值观、道德指标
8. 基金公司
9. 市场分析
10. 市场展望

---

### 十一、R9Alpha 研基结论

| 评估维度 | 评分标准 |
|---------|---------|
| 核心持仓持续性 | 核心重仓持有时间 |
| 行业轮动节奏 | 切换是否顺应趋势 |

| 道德与价值观 | 限购/发行时点 |
| 综合评分 | 1-5星 |

## 数据来源优先级

1. **本地持仓 CSV**：`{fundcode}_{YYYY}Q{Q}_holdings.csv`
2. **本地 JSON 数据**：`fund_{fundcode}_complete_data.json`
3. **本地风险收益 Excel**：`最新风险收益指标-({fundcode}.OF).xlsx`
4. **本地投资备忘录 Markdown**：`{基金名}_{fundcode}_投资研究备忘录.md`
5. **Wind API**：通过 `WindPy` 提取净值、风险指标、持有人结构、市盈率等
6. **网络公开资料**：季报访谈、基金经理观点、中国基金报、证券时报等
