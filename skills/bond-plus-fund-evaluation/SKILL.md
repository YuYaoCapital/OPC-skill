# 固收+基金产品评价 Skill

## 概述

本skill用于系统评价固收+基金产品（包括二级债基、偏债混合、稳健型FOF等），重点关注债券底仓稳健性和权益增强有效性。

## 评价框架

### 1. 基本信息

收集基金基础数据：
- 基金名称、代码、成立日期
- 基金类型（二级债基/偏债混合/灵活配置等）
- 管理规模、基金份额变动
- 业绩比较基准
- 费率结构（管理费、托管费、申购赎回费）

### 2. 基金经理

分析基金经理背景：
- 现任基金经理履历（任职日期、管理年限）
- 历史管理产品业绩
- 投资风格与理念
- 团队稳定性（是否频繁更换）
- 固收+产品管理经验

### 3. 收益指标

核心收益数据：
| 指标 | 说明 |
|------|------|
| 区间收益率 | 近1月/3月/6月/1年/3年/成立以来 |
| 年化收益率 | 几何年化收益 |
| 超额收益 | 相对业绩基准的超额 |
| 同类排名 | 在固收+基金中的分位 |
| 夏普比率 | 风险调整后收益 |
| Calmar比率 | 年化收益/最大回撤 |

**固收+基金关注重点**：年度正收益概率、滚动持有1年盈利概率

### 4. 风险控制指标（核心重点）

固收+基金的核心竞争力在于风险控制：

| 指标 | 关注要点 |
|------|----------|
| **最大回撤** | 历史最大回撤幅度 |
| **⭐最大回撤恢复天数** | 从最低点到修复的交易日数，**重点突出** |
| 回撤频率 | 年度回撤次数 |
| 波动率 | 年化标准差 |
| 下行标准差 | 仅计算负收益波动 |
| 索提诺比率 | 超额收益/下行标准差 |
| VaR(95%) | 风险价值 |

**最大回撤恢复天数分析框架**：
- 统计历次重大回撤的修复时间
- 对比同类平均修复天数
- 分析修复速度背后的原因（市场因素/调仓能力）
- **输出格式**：表格展示历次回撤的起止日期、回撤幅度、恢复天数

### 5. 收益归因分析（债底+权益增强分开）

#### 债券部分分析（底仓稳健性）

| 维度 | 分析内容 |
|------|----------|
| 久期管理 | 组合久期水平、久期调整时机 |
| 信用结构 | 信用债占比、信用等级分布 |
| 杠杆水平 | 杠杆率变动、回购融资情况 |
| 券种配置 | 利率债/信用债/可转债占比 |
| 个券选择 | 重仓债券、城投债/产业债分布 |

#### 权益增强部分分析（"+"的部分重点突出）

| 维度 | 分析内容 |
|------|----------|
| 股票仓位 | 股票仓位中枢、择时调整 |
| 行业配置 | 重仓行业、行业轮动能力 |
| 个股选择 | 重仓股、换手率 |
| 可转债增强 | 可转债仓位、转股溢价率偏好 |
| 打新收益 | 参与频率、收益贡献 |
| 衍生品对冲 | 国债期货、股指期货使用 |

**归因输出格式**：
- Brinson归因：资产配置效应 + 个股选择效应
- 时间序列归因：债券利息收入 vs 资本利得 vs 股票收益
- **突出权益增强部分**：单独计算"+"部分的收益贡献和胜率

### 6. 重仓股调仓有效性

评估权益端调仓能力：

| 分析维度 | 具体内容 |
|----------|----------|
| 调仓频率 | 季度换手率、持股周期 |
| 调仓胜率 | 新增/剔除个股后续表现 |
| 重仓股收益 | 前十大重仓股季度贡献 |
| 行业调整 | 行业配置变动及效果 |
| 时机把握 | 加仓/减仓时点准确性 |

**有效性评估方法**：
- 对比调仓前后个股/行业收益
- 新增个股持有期收益 vs 剔除个股后续收益
- 加权计算调仓决策的胜率

### 7. 新闻资讯收集（债券与权益分开）

#### 债券端新闻关注点
- 货币政策（MLF/降准/降息）
- 信用风险事件（违约、评级调整）
- 利率债发行与流动性
- 基金经理对债市的观点

#### 权益端新闻关注点
- 重仓个股公告与研报
- 行业政策变化
- 市场风格轮动
- 基金经理对股市的观点

**执行方式**：使用WebSearch分别搜索"基金代码+债券"和"基金代码+股票/重仓股"

### 8. 理财经理沟通流程

标准化沟通话术框架：

#### 开场白（客户画像匹配）
```
"根据您的风险偏好，这款固收+产品比较适合您..."
```

#### 产品介绍（突出核心卖点）
- 债券底仓的安全性
- 权益增强的历史效果
- **重点强调**：回撤控制能力和修复速度

#### 风险揭示
- 可能的回撤幅度
- 持有期建议（建议1年以上）
- 不适合短期频繁交易

#### 异议处理
| 客户异议 | 应对话术 |
|----------|----------|
| "收益不如股票基金" | 解释风险收益比，强调持有体验 |
| "担心亏钱" | 展示历史正收益概率、回撤修复速度 |
| "现在是不是好时机" | 分析当前股债性价比 |

#### 促成与跟进
- 建议配置比例
- 定投 vs 一次性投入
- 后续跟踪安排

### 9. 基金经理公开访谈收集

收集渠道：
- 基金公司官方公众号/视频号
- 天天基金、蚂蚁财富的基金经理说
- 券商研报中的路演纪要
- 财经媒体专访（中国基金报、证券时报等）

**收集要点**：
- 投资框架与理念
- 对当前市场的看法
- 后续操作策略
- 风控措施说明

## 输出格式

生成完整评价报告，包含：

1. ** Executive Summary ** - 一页纸核心结论
2. ** 详细分析 ** - 按上述9个模块展开
3. ** 投资建议 ** - 适合人群、配置比例、持有期建议
4. ** 风险提示 ** - 主要风险点

## 执行流程

当用户请求评价某只固收+基金时：

1. **确认基金代码** - 获取准确的基金代码和名称
2. **收集基础数据** - 使用Wind/同花顺iFinD等工具获取数据
3. **计算风险指标** - 重点计算最大回撤恢复天数
4. **收益归因** - 拆解债券和权益贡献
5. **搜索新闻** - 分开搜索债券和权益相关资讯
6. **整理访谈** - 收集基金经理最新观点
7. **生成话术** - 针对该产品特点准备沟通要点
8. **输出报告** - 按模板生成完整评价文档
